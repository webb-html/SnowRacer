проект представляет игру, где надо скатыватся по склону, убегая от монстра
игра бесконечная, карта зацикленная

в игре используется шрифт, в стиле игры, он поддерживает только английские буквы

на главном экране есть кнопка настроек 
	там можно настроить громкость фоновой музыки, вкл/выкл фоновую музыку (кнопка switch music), выбрать конкретных из треков

есть кнопка рекордов
	там можно посмотреть свои рекорды

над кнопкой старта игры есть поле ввода
	cюда можно ввести имя под которым запишется рекорд, если не трогать то имя будет incognito

кнопка старта игры перекидывает на экран выбора сложности
	на разных сложностях монстр становится быстрее, и т.п, на более чем сложнее тем быстрее зарабатываются очки


после выбора сложности начинается игра (если ошибка см. примечание)
	лыжник скатывается вниз сам, постепенно набирая скорость
	A, D или стрелки для передвижения в бок
	W или стрелка для тормоза скорости скатывания

	на пути встречаются препятствия при столкновении с ними скорость теряется
	кроме них есть трамплины - они ускоряют
	и сети они замедляют, но через них можно пройти (если хватит скорости)

	если монстр находится далеко (насколько зависит от сложности) он ускоряется до скорости игрока

	игра длится бесконечно

	если монстр догоняет игра заканчивается

после этого очки записываются в рекорды

примечание 1
(пытался поделится с другом .exe и выяснил что,)
при первом запуске карты могут произойти проблемы с загрузкой карты, 
в таком случае я нашел только одно решение: открыть файл trace.tmx через Tiled, будет сообщение, что некоторые файлы не найдены, список файлов
нужно нажать на единственный файл в списке, снизу загорится кнопка "открыть набор тайлов", после нажатия на нее, откроется идентичное сообщение - некоторые файлы не найдены,
нужно снова нажать на этот файл, вместо кнопки открыть набор файлов будет кнопка "Расположение файла...", после нажатие на нее откроется диалоговое окно выбора файла, надо
выбрать файл из папки проекта - tilemap.png, нажать на кнопку открыть, после этого нажать Ctrl + S. Готово!

почему-то со старта Tiled пытается найти тайлсет в другой дериктории, после того как первый раз ему объяснить файл будет работать нормально

это придется отдельно делать и для .exe и для .py файла
